# Virtual File System

This project is on making a Unix-like Virtual File System

## Usage:
```bash
git clone https://github.com/Preet2204/Virtual-File-System.git
cd Virtual-File-System
```

For Compilation:
```bash
g++ main.cpp cli/cli.cpp disk/disk.cpp fs/fs.cpp -o vfs.out
```

For Execution:
```bash
./vfs.out
```
